> XMLStarlet is a set of command line utilities (tools) which can be
> used to transform, query, validate, and edit XML documents and files
> using simple set of shell commands in similar way it is done for
> plain text files using UNIX grep, sed, awk, diff, patch, join, etc
> commands.

XMLStarlet is a venerable Swiss Army Knife for carving up XML. I was
having some issues with [`xmllint`][3] checking [Discogs Data][2] XML
files for well-formedness and stumbled upon XMLStarlet. On initial
test drive, seemed to be doing a bit better with the large size of the
Discogs files.
