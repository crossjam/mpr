Link parkin’. A couple of publicly available modules for taking
personal data and stuffing it into an SQLite database, congruent with
Simon Willison’s [Dogsheep initiative][1]

> Dogsheep is a collection of tools for personal analytics using
> SQLite and Datasette.

* [LastFM to SQLite, Jacob Kaplan-Moss version][2]
* [lastfm-to-sqlite, PyPI edition][3]
* [overcast-to-sqlite][4]
* [pinboard-to-sqlite][5] _Added 2022-07-17_
