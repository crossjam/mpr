The Wes McKinney led [**pandas**][1] project has just [hit the 0.8 release point][2]. I’ve been fiending for an excuse to use pandas and now an opportunity at work has popped up to do some timeseries-ish analysis. If for nothing else I’m looking forward to being able to easily [generate sequences of timestamps][3]:

> New DatetimeIndex class supports both fixed frequency and irregular time series. Replaces now deprecated DateRange class
